266 // [experimental]
267 // Used to activate or deactive the Mempurge feature (memtable garbage
268 // collection). (deactivated by default). At every flush, the total useful
269 // payload (total entries minus garbage entries) is estimated as a ratio
270 // [useful payload bytes]/[size of a memtable (in bytes)]. This ratio is then
271 // compared to this `threshold` value:
272 // - if ratio<threshold: the flush is replaced by a mempurge operation
273 // - else: a regular flush operation takes place.
274 // Threshold values:
275 // 0.0: mempurge deactivated (default).
276 // 1.0: recommended threshold value.
277 // >1.0 : aggressive mempurge.
278 // 0 < threshold < 1.0: mempurge triggered only for very low useful payload
279 // ratios.
280 // [experimental]
