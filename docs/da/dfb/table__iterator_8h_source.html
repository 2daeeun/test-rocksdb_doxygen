14// `TableReader::NewIterator`. The `InternalIterator` should be allocated with
15// the default allocator, not on an arena.
16// NOTE: Callers should ensure the wrapped `InternalIterator*` is a valid
17// pointer before constructing a `TableIterator` with it.
18class TableIterator : public Iterator {

24 }
