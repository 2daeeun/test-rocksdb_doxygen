상세한 설명

IndexBuilder for two-level indexing. Internally it creates a new index for each partition and Finish then in order when Finish is called on it continiously until Status::OK() is returned.

The format on the disk would be I I I I I I IP where I is block containing a partition of indexes built using ShortenedIndexBuilder and IP is a block containing a secondary index on the partitions, built using ShortenedIndexBuilder.
